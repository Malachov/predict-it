predictit.data_prep module¶

Module for data preprocessing. Contain functions for removing outliers, NaN columns or function for making inputs and outputs from time series if you want to see how functions work - working examples with printed results are in tests - visual.py.

Remove values far from mean - probably errors.

Parameters

  • predicted_column_index (int, optional) – Column that is predicted. Defaults to 0.

  • threshold (int, optional) – How many times must be standard deviation from mean to be ignored. Defaults to 3.
